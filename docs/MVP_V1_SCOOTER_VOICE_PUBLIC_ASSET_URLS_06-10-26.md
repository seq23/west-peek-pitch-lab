# MVP v1 Scooter Voice Public Asset URLs — 06-10-26

## Purpose

D-ID cannot fetch local repo paths. The Scooter source image and Scooter voice audio must be available at public HTTPS URLs after the Pitch Lab site deploys.

## Added assets

```txt
public/assets/avatar/scooter-voice-only.m4a
public/assets/avatar/scooter-voice-only.mp3
```

The `.mp3` file was generated from the supplied `.m4a` source for broader provider compatibility.

## Expected deployed URLs

Replace `<PITCH_LAB_DEPLOY_OR_CUSTOM_DOMAIN>` with the deployed Pitch Lab domain.

```txt
https://<PITCH_LAB_DEPLOY_OR_CUSTOM_DOMAIN>/assets/avatar/scooter-voice-only.m4a
https://<PITCH_LAB_DEPLOY_OR_CUSTOM_DOMAIN>/assets/avatar/scooter-voice-only.mp3
```

For D-ID live proof, prefer the MP3 URL unless provider testing proves M4A works reliably.

## D-ID usage

D-ID should receive:

```txt
source_url = public URL for /assets/avatar/scooter-avatar-source.png
audio_url  = public URL for /assets/avatar/scooter-voice-only.mp3
```

The final talking-avatar MP4 is generated by D-ID and returned as a provider result URL.

## Security

These audio files are public static assets once deployed. They are not API secrets.
